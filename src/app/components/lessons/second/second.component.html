Steps to create a new Angular project

    • ng new project_name

    • You will see the following question

    Which stylesheet format would you like to use? (Use arrow keys)

    With the following options

    > CSS[ https : //deve10per.mozi11a.org/docs/Web/CSS]

    Sass (SCSS)[ https://sass-lang.com/documentation/syntax#scss]

    Sass (Indented) [ https://sass-lang.com/documentation/syntax#the-indented-syntax]

    Less [ http://lesscss.org]

    • A second question will be asked

    ? Do you want to enable Server-Side Rendering (SSR) and Static Site Generation (G/Prerendering)?

    • cd project_name

    Now you are inside your Angular project directory

    • ng serve

    The Angular CLI will provide a local development server allowing you to view your project in the browser and automatically refresh when changes occur in the files.

    Por padrão o Angular CLI disponibilizará o servidor http://localhost:4200/
